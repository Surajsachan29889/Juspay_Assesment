# Frontend - eCommerce Dashboard

## Quick Start

```bash
npm install
npm start
```

## Available Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm test` - Run tests

## Environment Variables

Create a `.env` file in the root directory:

```env
REACT_APP_API_URL=your_api_url_here
```
